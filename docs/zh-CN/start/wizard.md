# 手动向导（CLI）

`openclaw onboard` 现在是**手动 / 高级**设置路径。

大多数本地 CodexPlusClaw 安装请优先使用：

```bash
openclaw setup --one-click
```

以下情况再使用手动向导：

- 远程 Gateway
- 高级认证或提供商流程
- 逐步配置渠道
- 非交互式脚本化向导流程

```bash
openclaw onboard
```
