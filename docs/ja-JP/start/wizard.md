# オンボーディングウィザード（CLI）

`openclaw onboard` は**手動またはリモート用の詳細ウィザード**です。
通常のローカルセットアップなら `openclaw setup --one-click` が推奨です。

```bash
openclaw onboard
```

このウィザードは、以下のような場面で使います。

- リモートGatewayを接続したい
- すべての設定を手動で確認したい
- チャネルや認証を段階的に設定したい
